### **1. Logic and Predicates**

- **Truth Values:** **$⊤$** represents true, and **$⊥$** represents false.
- **Connectives:**
  - **$∧$** (And), **$∨$** (Or), **$¬$** (Not).
  - **$⇒$** (Implies): Note that $p ⇒ q$ is considered true if $p$ is false.
  - **$⇔$** (If and only if/Iff): Used to denote logical equivalence between two propositions.
- **Quantifiers:** **$∀$** (Universal/“For all”) and **$∃$** (Existential/“There exists”).
- **Predicates:** Written as **$P(n)$**, representing a function that returns a proposition based on its input.

### **2. Set Theory and Relations**

- **Membership and Equality:** **$∈$** (Member of), **$∉$** (Not a member of). Two sets are equal (**$A = B$**) if they share the exact same elements.
- **Subsets:** **$A ⊆ B$** means every element in $A$ is also in $B$. **$A ⊈ B$** means it is not a subset.
- **Operations:**
  - **$∅$**: The empty set.
  - **$A ∪ B$** (Union), **$A ∩ B$** (Intersection), **$A \setminus B$** or **$A - B$** (Set difference).
  - **$A \times B$**: Cartesian product, the set of all pairs $(x, y)$.
  - **$℘(S)$** or **$2^S$**: Power set, the set of all subsets of $S$.
  - **$Fin(S)$**: The set of all finite subsets of $S$.
- **Relations and Equivalence:**
  - **$xRy$**: Denotes that $x$ is related to $y$ under relation $R$.
  - **$[x]_R$**: The equivalence class of $x$, containing all elements $y$ such that $xRy$.
  - **$A/R$**: The quotient set, which is the set of all equivalence classes.
  - **$p ∼ r$**: Often used to denote that two states in an automaton are equivalent.

### **3. Functions**

- **Types:** **$A → B$** denotes a total function, while **$A ⇀ B$** denotes a partial function.
- **Components:** **Domain** (input set), **Codomain** (potential output set), and **Image** (actual set of outputs).
- **Operations:** **$id$** is the identity function, and **$f ∘ g$** is the composition of two functions.

### **4. Alphabets, Strings, and Languages**

- **Strings:**
  - **$\Sigma$**: A finite, non-empty alphabet.
  - **$\varepsilon$**: The empty string (also written as nil or [ ]).
  - **$uv$**: The concatenation of two strings $u$ and $v$.
  - **$|w|$**: The length of the string $w$.
  - **$w^n$**: The string $w$ repeated $n$ times.
- **Language Sets:**
  - **$\Sigma^*$**: The set of all possible strings over alphabet $\Sigma$ (Kleene star).
  - **$\Sigma^+$**: The set of all non-empty strings over alphabet $\Sigma$.
  - **$\Sigma^n$**: The set of all strings of exactly length $n$.
  - **$LM$**: Concatenation of languages $L$ and $M$.
  - **$L^n$**: Language $L$ concatenated with itself $n$ times.

### **5. Automata and Regular Expressions**

- **Automata Tuples:** Typically a 5-tuple **$(Q, \Sigma, \delta, q_0, F)$** representing states, alphabet, transition function, start state, and accepting states.
- **Transition Functions:**
  - **$\delta$**: Direct transition.
  - **$\hat{\delta}$**: The extended transition function for strings.
  - **$\varepsilon\text{-closure}(q)$**: The set of states reachable from $q$ using only $\varepsilon$-transitions.
- **Regular Expression Syntax:**
  - **$\emptyset$**: Represents the empty language.
  - **$e_1 + e_2$** (or $e_1 | e_2$): Alternation (Union).
  - **$e_1 e_2$**: Sequencing (Concatenation).
  - **$e^*$**: Kleene star; **$e^+$** is an abbreviation for $ee^*$.

### **6. Context-Free Grammars (CFGs)**

- **CFG Tuple:** A 4-tuple **$(N, \Sigma, P, S)$** representing nonterminals, terminals, productions, and the start symbol.
- **Derivations:**
  - **$A → \alpha$**: A production rule.
  - **$\Rightarrow$**: A single derivation step.
  - **$\Rightarrow^*$**: A derivation in zero or more steps.
  - **$\Rightarrow_{lm}$**: A leftmost derivation step.
- **Parse Trees and Inference:**
  - **$P(G, A)$**: The set of all parse trees for grammar $G$ rooted at nonterminal $A$.
  - **$node(A, ts)$**: A constructor for a parse tree node with nonterminal $A$ and a list of children $ts$.
  - **$yield(t)$**: The terminal string formed by the leaves of parse tree $t$.
  - **$L(G, A)$**: The language generated by a specific nonterminal $A$.
